# Art Gallery

Art Gallery is a bilingual digital gallery built with Next.js. It was created as a challenge project for the Majan Programming Club.

## Features

- English and Arabic localized routes
- RTL layout support for Arabic
- Gallery browsing and artwork detail pages
- Responsive layouts for mobile and desktop
- Custom logo and browser icon
- Like interaction for artworks

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

Useful routes:

- English home: `/en`
- Arabic home: `/ar`
- English gallery: `/en/gallery`
- Arabic gallery: `/ar/gallery`

## Scripts
